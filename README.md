# Customer_behavior_analysis
This project analyzes customer shopping behavior using a transactional dataset containing 3,900 customer purchases. The objective is to identify customer purchasing patterns, product preferences, subscription trends, and revenue insights through Python, SQL, and Tableau.

# 🛍️ Customer Shopping Behavior Analysis

## 📖 Overview

This project is an end-to-end **Data Analytics** project that analyzes customer shopping behavior to uncover valuable business insights. The project covers the complete analytics workflow, including data preprocessing, exploratory data analysis (EDA), SQL-based business analysis, interactive dashboard creation in Tableau, and report preparation.

The goal is to help businesses understand customer purchasing patterns, improve decision-making, and identify opportunities for growth.

---

## 📂 Dataset

The dataset contains customer shopping transaction data with information such as:

- Category
- Item Purchased
- Purchase Amount
- Review Rating
- Subscription Status
- Payment Method
- Shipping Type
- Discount Applied
- Previous Purchases
- Purchase Frequency
- Season
- Location

## 📋 Project Workflow

### 1. Data Loading
- Imported the dataset using Python.
- Inspected data types and dataset structure.

### 2. Exploratory Data Analysis (EDA)
- Checked missing values.
- Generated summary statistics.
- Explored customer demographics.
- Analyzed purchase behavior.
- Identified important trends.

### 3. Data Cleaning
- Handled missing values.
- Renamed columns for consistency.
- Created new features such as Age Group.
- Improved overall data quality.

### 4. SQL Analysis (MySQL Server)
Performed SQL queries to answer business questions including:

- Revenue by Gender
- Revenue by Age Group
- Customer Segmentation
- Subscription Analysis
- Shipping Type Comparison
- Top Products
- Discount Analysis
- Customer Purchase Trends

### 5. Dashboard Development (Tableau)
Created an interactive Tableau dashboard to visualize key insights, including:

- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Sales by Age Group
- Subscription Status Distribution
- Average Purchase Amount
- Average Review Rating
- Customer Count
- Interactive Filters

### 6. Report Preparation
Prepared a detailed project report summarizing:

- Data Cleaning
- EDA
- SQL Analysis
- Dashboard Insights
- Business Recommendations

### 7. Presentation
Created a professional project presentation using **Gamma AI** to showcase the complete analytics workflow and findings.

## 📈 Results

Some key insights from the analysis include:

- Clothing category generated the highest revenue.
- Young Adult customers contributed the highest sales.
- Non-subscribers represented the largest customer segment.
- Purchase behavior varies across age groups.
- Customer segmentation can support targeted marketing strategies.

## ▶️ How to Run

1. Clone this repository.

```bash
git clone https://github.com/your-username/Customer_Behavior_Analysis.git
```

2. Open the Python project and install the required libraries.

```bash
pip install pandas numpy matplotlib mysql-connector-python
```

3. Load the dataset into Python.

4. Run the data cleaning and EDA scripts.

5. Import the cleaned data into MySQL Server.

6. Execute the SQL queries.

7. Open the Tableau workbook to explore the dashboard.
